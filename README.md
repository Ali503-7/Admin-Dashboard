# Admin-Dashboard

## Installation

To install the necessary node modules, run the following command:

```bash
npm i
```

## Running the Project

To run the project, use the following command:
```bash
npm run dev
```
## Built with
- React.js
  - react-router-dom
  - [atom](https://github.com/hassanzohdy/mongez-react-atom) (state management tool)
